Adivinador


Akinator clone. Fourth semester university project for Data Structures II course. Program meant to guess any video game/video game series, given answers to questions. Also, learns new questions and answers.

The purpose of this assignment was to understand and apply trees, binary trees, binary search trees, and binary decision diagrams.

Based on Akinator, the application asks the user questions, selected from a question list, in an attempt to guess the video game or video game series the user is thinking. Once it believes to have reached the correct answer, it asks for confirmation.

The program selects questions by looking for an unanswered question which would reduce the number of possible answers by half, or closest to half.

If the program guesses correctly, it adds the answers given by the user, that were previously unknown, and adjusts the answers known with the ones given by the user to improve the chances of getting the game right in the future.

However, if the program guesses incorrectly, it keeps asking questions in an attempt to eventually get the right answer, however the user has already won. The user's score is added to the scoreboard.

If the video game the user thought of hasn't been added, the program asks the user to add it, as well as give the possibility for a question to be added. This answer is checked for similiarities with previously added video games to avoid duplicates.

Worked on March 2017 (Approximately one weekend, and a week of on and off bug fixing and tweaking).
