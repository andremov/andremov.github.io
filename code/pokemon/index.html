Pokemon

Pokemon clone. Third semester university project for Object-Oriented Programming course. Capture, collect, battle, and level up Pokemon. Has an expandable map, and its respective region builder.

Gallery

The purpose of this assignment was to learn the Java programming language, the NetBeans IDE, as well as Object-Oriented Programming.

Based on Pokemon, the application drops the player on a small map inhabited by all first generation Pokemon, available for capture, collection, and battle.

Special moves were not coded, making battles very simplistic.

The Shop, PC, PokeDex and Pokemon Center functions can be accessed through the pause menu.

Has a region builder that can be accessed through 'cheats', and allows for the map to be changed or expanded.

All Pokemon's stats are close to, if not, correct. Item and trade evolutions were changed to regular level up evolutions.

Worked from August 2016 to October 2016 (2 months).
