Bomberman

Super Bomberman 2 clone. Fourth semester university project for Data Structures II course. Local Area Network multiplayer and simplified version of Super Bomberman 2's Battle Mode.

Gallery

The purpose of this assignment was to learn ports and sockets, therefore allowing the connection of multiple computers in the same Local Area Network.

Based on Super Bomberman 2, the application allows for a close experience of playing Super Bomberman 2's Battle Mode, over simplified.

Allows up to 4 players: one host, and three connections.

Allows changing color of the player's character.

Counts the number of times a player has won.

Players can place bombs, which can be used to destroy blocks, or other players.

Destroyed blocks can spawn power ups that can be picked up to enhance a player's abilities, like the length of the player's bombs' explosion, or the number of bombs the player can place.

Worked from April 2017 to May 2017 (6 weeks).
